# 🛍️ Swiftly

A full-stack e-commerce marketplace where users can browse and buy accessories, and sellers can list their products for sale.

## 🚀 The Team
- **Backend Developer:** Niraj Mishra
- **Frontend Developer:** Yousuf

## 🛠️ Tech Stack
- **Backend:** Python, FastAPI
- **Database:** PostgreSQL
- **Frontend:** React/HTML/CSS

## 📋 Features
- **User Accounts:** Separate roles for Buyers and Sellers.
- **Product Listings:** Sellers can create, update, and delete item listings.
- **Marketplace:** Buyers can browse categories and view item details.
- **Search/Filter:** (Optional) Find accessories by price or type.
